🔧 修复列名错误

解决 draw_stats_view 中的 created_at 列不存在问题

❌ 错误信息

错误代码： 42703

错误描述： column "created_at" does not exist

ERROR: 42703: column "created_at" does not exist LINE 7: COUNT(CASE WHEN created_at >= CURRENT_DATE THEN 1 END) as today_draws, ^

问题原因： 在 draw_history 表中，时间戳列名是 draw_date，而不是 created_at

✅ 解决方案

需要重新创建 draw_stats_view 视图，使用正确的列名 draw_date 和 prize_won

-- ===================================================== -- 修复抽奖统计视图 - 解决列名错误问题 -- ===================================================== -- 1. 删除现有的错误视图 DROP VIEW IF EXISTS draw_stats_view; -- 2. 创建正确的抽奖统计视图 CREATE OR REPLACE VIEW draw_stats_view AS SELECT COUNT(*) as total_draws, COUNT(DISTINCT user_phone) as unique_participants, COUNT(CASE WHEN prize_won != '谢谢参与' THEN 1 END) as winning_draws, COUNT(CASE WHEN draw_date >= CURRENT_DATE THEN 1 END) as today_draws, MAX(draw_date) as latest_draw, ROUND( COUNT(CASE WHEN prize_won != '谢谢参与' THEN 1 END)::DECIMAL / NULLIF(COUNT(*), 0) * 100, 2 ) as win_rate FROM draw_history; -- 3. 验证视图创建成功 SELECT 'draw_stats_view 创建成功' as status; -- 4. 测试视图查询 SELECT * FROM draw_stats_view;

🔍 列名对照表

draw_history 表的正确列名：

  • draw_date - 抽奖时间（不是 created_at）
  • prize_won - 中奖奖品（不是 prize_name）
  • user_phone - 用户手机号
  • user_name - 用户姓名
